# Plain's Paris Express + Twilio server

This project was created with an Express in the `server` directory. The server is proxied via the `proxy` key in `package.json`.

## Using this project

1. Clone the project, change into the directory and install the dependencies.

   ```bash
   git clone git@github.com:arjangupta/plains-paris-text-server.git
   cd plains-paris-text-server
   npm install
   ```

2. Create a `.env` file for environment variables in your server.

   ```bash
   touch .env
   ```

3. Start the server

   You can start the server on its own with the command:

   ```bash
   npm run server
   ```
